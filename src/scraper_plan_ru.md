# План скрипта для скрапинга

## Система меню
1. Ввести адрес сайта.
2. Ввести тег.
3. Выход.

## Организация файлов
- Директория для вывода: `output_samples/Проект R`
- Папка сайта: `site_name` (пробелы заменяются на нижние подчёркивания)
- Папка тега: `tag_name` (пробелы заменяются на нижние подчёркивания)
- Выходные файлы: `.md` и `.json`

## Формат вывода
- Файл `.md`: содержит заголовки страниц и контент.
- Файл `.json`: содержит структуру данных страницы.

## Опция повторного использования
- Сохранять последний введённый адрес сайта для повторного использования.
- Разрешать корректировку тега.

## Интеграция
- Использовать существующую структуру `src/rss_fetcher.py`.
- Добавить обработку ошибок для некорректных входных данных.

## Тестирование
- Проверить создание папок.
- Протестировать на примере тестового сайта и значений тега.

## Документация
- Включить инструкцию по использованию.
